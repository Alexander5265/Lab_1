# Задание 3.

## Задача 2.

### Текст задачи.

Создайте собственные функции для выполнения основных операций над списками (добавление/
удаление/поиск элемента, сцепка двух списков, получение элемента по номеру).

### Алгоритм решения.

Все делаем через patter matching и кортежи. 
1. Функция добавления эл в конец списка. Рекурсией идем до конца списка, при достижении конца - добавляем элемент, затем возвращаем предыдущие значения.
2. Функция удаления эл. Также рекурсивно проверям список, эл != нужному эл - значит идем дальше. Если эл == нужному эл - удаляем его.
3. Функция посика эл в заданном списке. Обозначаем кортеж с условием, что если достигнут конец списка и эл не найден - то его нет. Если текущий эл нужный - то перемещаем его в конец, если нет - идем дальше.
4. Функция объединения списков. Проходит полностью по первому списку и при достижении конца списка заполняет его значениями второго списка, затем рекурсивно возвращает предыдущие значения первого списка. 
5. Функция получения элемента по индексу. Сразу обозначим, если достигли конца списка - то индекс был вне допустимого диапазона. Иначе идем по списку и рекурсивно возвращаем нужный элемент. 
6. Пишем main функцию. В ней задаем два готовых списка и опираясь на них по очерди вызываем функции, вставляя свои значения.
